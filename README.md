# Homebrew Tap for Loopwerk

This is the official [Homebrew](https://brew.sh) tap for tools by [Loopwerk](https://github.com/loopwerk).

## Installation

```
brew install loopwerk/tap/saga
```

## Available Formulae

| Formula | Description |
|---------|-------------|
| [saga](Formula/saga.rb) | A code-first static site generator written in Swift |
